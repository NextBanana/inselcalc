# ⚔️ Kampfinsel Kampfrechner

[![MIT License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

> **Inoffizielles Tool** — Dieses Projekt steht in keiner Verbindung zum Kampfinsel-Team.

Ein interaktiver Kampfrechner für [Kampfinsel](https://kampfinsel.com) — das kostenlose Strategie-Browsergame. Berechne Angriff, Verteidigung und Beute in Echtzeit, bevor du deine Flotte losschickst.

## Features

- Live-Berechnung von Gesamtangriff und -verteidigung
- Truppenkapazitätsprüfung mit Warnung bei Überschreitung
- Beuteberechnung (proportional auf Gold, Silber, Holz aufgeteilt)
- Anzeige fehlender Einheiten bei Niederlage

## Nutzung

Öffne einfach `kampfrechner.html` im Browser — keine Installation nötig.

## Deployment

### GitHub Pages

Push auf `main` deployt automatisch via GitHub Actions.  
GitHub Pages in den Repository-Einstellungen einmalig aktivieren (Source: **GitHub Actions**).
